# Cursor Website Recreation

This project is a recreation of the Cursor website, focusing on the visual design and layout of key sections.

## Recreated Sections

The following sections from the original website have been recreated:

1.  **Header**: Main navigation bar with logo and menu links.
2.  **Hero Section**: Introduction area with main headline and call-to-action.
3.  **Company Logos**: Display of companies using Cursor.
4.  **Agents**: "Agents turn ideas into code" feature section.
5.  **Tab Autocomplete**: "Magically accurate autocomplete" feature section.
6.  **Surfaces**: "In every tool, at every step" feature section.
7.  **Testimonials**: User reviews and feedback.
8.  **Frontier**: "Stay on the frontier" card grid showing models, indexing, and enterprise features.
9.  **Changelog**: Updates and release notes.
10. **Team**: Information about the Cursor team.
11. **Recent Highlights**: Blog posts and news items.
12. **CTA**: Final "Try Cursor now" call-to-action.
13. **Footer**: Site map and legal links.

## Design System

### Fonts

The project primarily uses the **Inter** font family, falling back to system sans-serif fonts.

- Family: `"Inter", -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`

### Colors

The design utilizes a dark theme with high-contrast text and specific accent colors.

**Backgrounds**

- Site Background: `#14120b`
- Card Background: `#1b1913`
- Main Background: `#0a0a0a`

**Text**

- Primary Text: `#ffffff`
- Muted Text: `#a3a3a3`
- Card Headings: `#edecec`
- Card Paragraphs: `#edecec99`

**Accents & Interactions**

- Link Color: `#ff4e00` (Orange)
- Link Hover: `#cc6a3d`
- Borders: `rgba(255, 255, 255, 0.15)`
- Border Hover: `rgba(255, 255, 255, 0.3)`
